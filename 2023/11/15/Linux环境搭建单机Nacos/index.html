Linux环境搭建单机Nacos

一、安装JDK

Nacos 依赖 Java 环境来运行，需要安装JDK。如果是从代码开始构建并运行Nacos，还需要为此配置 Maven环境。

# 查看jdk版本列表
yum -y list java*

# 选择需要的版本，下载安装
yum install -y java-1.8.0-openjdk.x86_64
# 不安装这个无法使用javac命令，会导致nacos启动不了，上面那个可装可不装
yum install -y java-1.8.0-openjdk-devel.x86_64

# 验证是否安装成功
java -version

# yum指令安装jdk会自动配置环境变量，可使用以下命令检查环境变量
echo $JAVA_HOME

# jdk默认安装到/usr/lib/jvm目录下

二、下载、安装Nacos

Nacos安装包下载地址

# 创建nacos文件夹
mkdir -m 777 -p /usr/local/nacos

# 将Nacos安装包上传到/usr/local/nacos文件夹下（或者可以使用wget指令拉取）

# 解压
tar -zxvf nacos-server-2.0.3.tar.gz

# 进入nacos的bin目录下
cd /usr/local/nacos/nacos/bin

# 启动nacos（此命令为单机启动nacos）
sh startup.sh -m standalone

开放Nacos默认端口8848

三、进入Nacos主页，确认是否启动成功

浏览器访问 http://ip地址:端口号/nacos 测试nacos是否启动成功

Nacos初始账号密码：
nacos
nacos
